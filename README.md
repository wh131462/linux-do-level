# linux.do 等级查询 - 增强版

linux.do 等级查询用户脚本，支持查看信任等级和升级进度。

## 功能特性

- 🎯 实时查询当前信任等级
- 📊 显示升级进度和要求
- 🎨 编辑式极简设计风格
- ⚡ 支持等级 0-3 的详细数据

## 安装

1. 安装浏览器扩展：[Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)
2. [点击安装脚本](https://greasyfork.org/zh-CN/scripts/你的脚本ID)
